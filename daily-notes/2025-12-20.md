### 2. 교차 곱으로 기울기 비교 (120875)

```python
# 나눗셈 → 교차 곱 변환 (부동소수점 오차 방지)
# (y2-y1)/(x2-x1) == (y4-y3)/(x4-x3)
# ↓
(y2 - y1) * (x4 - x3) == (y4 - y3) * (x2 - x1)
```

**4점에서 2쌍 조합:** (0-1, 2-3), (0-2, 1-3), (0-3, 1-2) → 3가지

### 3. 겹치는 구간 - 두 가지 접근법 (120876)

**v1. 스위핑:**
```python
for i in range(left, right):
    overlap_count = sum(1 for a, b in lines if a <= i < b)
```
