## 🎯 핵심 학습 내용

### 1. 시퀀스(리스트/튜플) 사전순 비교

Python의 리스트/튜플은 **lexicographic(사전순) 비교**를 수행한다.

```python
# 날짜 비교 - 복잡한 조건문 대신 한 줄로!
[2021, 12, 28] < [2021, 12, 29]  # True
# 비교 순서: 2021==2021 → 12==12 → 28<29 → True
```

**활용 패턴:**
- 날짜 비교: `[년, 월, 일]`
- 버전 비교: `[major, minor, patch]`
- 다중 조건 정렬: `sorted(items, key=lambda x: (조건1, 조건2))`

→ 상세 내용: `notes/sequence-comparison.md`

### 2. 리스트 곱셈의 참조 복사 특성

```python
# v1: 같은 문자열을 k번 새로 생성 → O(n * m * k²)
for _ in range(k):
    answer.append("".join(c * k for c in line))

# v2: 한 번 생성 후 참조만 복사 → O(n * m * k)
answer += ["".join(c * k for c in line)] * k
```

**핵심:** `[item] * k`는 같은 객체를 k번 **참조**하는 리스트를 만든다 (복사 X).

### 3. `in` 연산자 활용 패턴

오늘 4문제에서 동일한 패턴 사용:

```python
# 공통 패턴: 포함 여부를 0/1로 반환
return int(target in container)
```

| 자료구조 | `in` 연산 시간복잡도 |
|----------|---------------------|
| list | O(n) |
| str (부분 문자열) | O(n * m) |
| set | O(1) |
| dict (키 검색) | O(1) |

### 4. 필터 + join 패턴

```python
# 조건에 맞는 문자열만 결합
"".join(s for s in str_list if ex not in s)
```

3가지 기법 조합:
1. Generator expression - 메모리 효율적 반복
2. 조건부 필터링 - `if` 절
3. join - 문자열 결합
