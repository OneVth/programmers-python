## 🎯 핵심 학습 내용

### 1. list() 복사로 원본 보존 (문제 #181883)

```python
def solution(arr: list[int], queries: list[list[int]]) -> list[int]:
    answer = list(arr)  # 얕은 복사 — 원본 보존
    for query in queries:
        for i in range(query[0], query[1] + 1):
            answer[i] += 1
    return answer
```

`list(arr)` vs `arr[:]` vs `arr.copy()` — 모두 얕은 복사, 취향 차이

### 2. zip 병렬 순회 필터링 (문제 #181885)

```python
# 두 리스트를 동시에 순회하며 조건 필터링
[x for x, b in zip(todo_list, finished) if not b]
```

`zip`은 **가장 짧은 리스트 기준**으로 순회 — 길이가 다를 수 있다면 `zip_longest` 사용

### 3. 슬라이싱 step 패턴 정리 (문제 #181886, 181887, 181888)

```python
names[::5]      # 매 5번째 원소 추출 (0, 5, 10, ...)
num_list[::2]   # 홀수 번째 (1st, 3rd, 5th...) = 인덱스 0, 2, 4
num_list[1::2]  # 짝수 번째 (2nd, 4th, 6th...) = 인덱스 1, 3, 5
num_list[::n]   # 매 n번째 원소 일반화
```

**슬라이싱 `[start:stop:step]`의 세 번째 인자(step)가 핵심:**
- step만 지정: `[::k]` → 처음부터 k칸 간격
- start도 지정: `[1::k]` → 인덱스 1부터 k칸 간격

### 4. for+break vs while (문제 #181884)

```python
# for+break: 리스트 순회 중 조건 충족 시 중단
answer = 0
for i in numbers:
    if answer > n:
        break
    answer += i
```

`for+break`는 "리스트를 순회하되 조건에서 멈추기"에 적합, `while`은 "조건이 참인 동안 반복"에 적합
