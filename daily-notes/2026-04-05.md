### #250129 - 산책

**핵심 아이디어:** 2D 좌표 누적(displacement)  
N/S → north 증감, E/W → east 증감 후 `[east, north]` 반환

```python
for i in route:
    if i == "N": north += 1
    elif i == "S": north -= 1
    elif i == "E": east += 1
    elif i == "W": east -= 1
```

**배운 점:** 방향을 `dict`로 매핑하면 분기 없이 처리 가능
```python
dirs = {'N':(0,1), 'S':(0,-1), 'E':(1,0), 'W':(-1,0)}
```

### #250130 - 저축

**핵심 아이디어:** 두 단계 while 루프 (70만원 기준 분리)

```python
while money < 70:

while money < 100:

```

**배운 점:** 단일 루프 + 삼항 표현으로도 동일하게 구현 가능
```python
rate = before if money < 70 else after
```

### #250131 - 나이 계산

**핵심 아이디어:** 한국식(+1) vs 연 나이(+0) 오프셋 차이

```python
if age_type == "Korea":
    answer = 2030 - year + 1
elif age_type == "Year":
    answer = 2030 - year
```

**배운 점:** Python에서 `bool`은 `int` 서브클래스 → 한 줄 표현 가능
```python
return 2030 - year + (age_type == "Korea")
```

### #250132 - 피타고라스의 정리

**핵심 아이디어:** `b² = c² - a²` 직접 계산 (디버깅 유형)

```python
b_square = c**2 - a**2
```

**배운 점:** 원본 C++ 버그는 `c - a` (단순 뺄셈) → `c*c - a*a`로 수정해야 함  
`c² - a² = (c-a)(c+a)` 인수분해로 버그 원인 파악 가능

### #250133 - 출력

**핵심 아이디어:** 타입에 따른 `+` 연산자 동작 차이

```python
string_msg = "Spring is beginning"  # str
int_val = 3                          # int: 3 + 10 = 13
string_val = "3"                     # str: "3" + "10" = "310"

return "\n".join([string_msg, str(int_val + 10), string_val + "10"])
```

**배운 점:**
- `int + int` → 산술 덧셈
- `str + str` → 문자열 연결 (연산자 오버로딩)
- Python은 암묵적 타입 변환 없음 → `str()` 명시적 변환 필요
