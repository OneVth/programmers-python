## 오늘 복습한 핵심 개념

### 1. 삼각 부등식과 range 길이 활용 (120868)

```python
# 두 변 a, b가 주어졌을 때 세 번째 변 x의 범위
# |a - b| < x < a + b

# 정수 개수 = range 길이로 한 줄 해결
len(range(max(sides) - min(sides) + 1, sum(sides)))
```

**핵심:** `range(start, end)`의 길이가 곧 정수 개수

### 2. 아나그램 판별 - Set vs 정렬 (120869)

```python
# v1. Set 비교 (중복 없을 때)
set(spell) == set(word)

# v2. 정렬 비교 (중복 있어도 가능)
sorted(spell) == sorted(word)
```

**차이점:**
| 방식 | 중복 처리 | 시간복잡도 |
|------|----------|-----------|
| Set | 무시됨 | O(n) |
| 정렬 | 보존됨 | O(n log n) |
