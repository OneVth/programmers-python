### 1. rindex() vs rfind() (문제 #181872)

```python
s = "abcabc"

s.rindex("abc")  # 3 (마지막 등장 위치, 없으면 ValueError)
s.rfind("abc")   # 3 (마지막 등장 위치, 없으면 -1)
```

| 메서드 | 못 찾을 때 | 용도 |
|--------|:--------:|------|
| `rindex()` | ValueError | 반드시 존재할 때 |
| `rfind()` | -1 | 존재 여부 불확실할 때 |

```python
# 패턴의 마지막 등장까지 슬라이싱
def solution(myString: str, pat: str) -> str:
    idx = myString.rindex(pat)
    return myString[:idx] + pat  # 또는 myString[:idx + len(pat)]
```

### 2. 조건부 문자 변환 패턴 (문제 #181873)

```python
# 방법 1: 리스트 컴프리헨션 + join
"".join([c.upper() if c == alp else c for c in my_string])

# 방법 2: replace (더 간결)
my_string.replace(alp, alp.upper())
```

`replace`가 더 간결하지만, 여러 조건이 필요할 때는 컴프리헨션이 유연함
