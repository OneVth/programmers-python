## 💡 오늘의 핵심 배움

### 유클리드 호제법에 max/min이 필요 없다!

```python
def gcd(a, b):
    while b != 0:
        a, b = b, a % b
    return a

# a < b일 때 첫 반복에서 자동 swap
# gcd(3, 12) → (12, 3) → (3, 0) → 3
```

### 삼각 부등식 범위 공식화

```python
# 두 변 a, b가 주어질 때, 세 번째 변 x의 범위
# |a - b| < x < a + b

# 가능한 정수 개수 = 2 * min(a, b) - 1
return 2 * min(sides) - 1
```

### sorted vs set 비교 시 주의점

```python
# set은 중복을 무시함!
set("aa") == set(['a'])  # True

# 따라서 길이 체크 필수
if len(word) == len(spell) and set(word) == spell_set:
```
