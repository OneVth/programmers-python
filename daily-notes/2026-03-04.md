## 🎯 핵심 학습 내용

### 1. 겹치는 부분 문자열 카운트 (문제 #181871)

```python
def solution(myString: str, pat: str) -> int:
    answer = 0
    idx = 0
    while (idx := myString.find(pat, idx)) != -1:
        answer += 1
        idx += 1  # +1로 겹치는 매치 허용
    return answer
```

**`str.count()` vs `find()` 루프:**

| 방식 | 겹치는 매치 | 예시 `"aaa"에서 "aa"` |
|------|:---------:|:----:|
| `str.count("aa")` | X | 1 |
| `find()` 루프 (idx += 1) | O | 2 |

### 2. `in` 연산자로 부분 문자열 필터링 (문제 #181870)

```python
# "ad"가 포함되지 않은 문자열만 추출
[s for s in strArr if "ad" not in s]
```

`in` 연산자는 리스트에서는 O(n), 문자열에서는 O(n*m)이지만 짧은 패턴에서는 충분히 빠름
