## 🎯 핵심 학습 내용

### 1. `math.prod()` — 리스트 원소의 곱

```python
from math import prod

prod([3, 4, 5, 2, 1])  # 120
```

- Python 3.8+에서 추가된 내장 함수 — `functools.reduce(operator.mul, lst)` 대비 훨씬 간결
- 대안: `functools.reduce(lambda a, b: a * b, num_list)`
- 빈 리스트에 대해 `prod([]) == 1` (항등원) — `sum([]) == 0`과 동일한 수학적 관례

### 2. `int(조건식)` — 불리언을 정수로 변환

```python
# Before: 조건부 return
if prod(num_list) < sum(num_list) ** 2:
    return 1
else:
    return 0

# After: int() 변환 관용 패턴
return int(prod(num_list) < sum(num_list) ** 2)
```

- Python에서 `bool`은 `int`의 서브클래스: `True == 1`, `False == 0`
- `int(True) == 1`, `int(False) == 0` — 이를 이용해 조건부 return을 한 줄로 압축
- "0 또는 1을 반환하라"는 문제에서 자주 등장하는 패턴

### 3. AM-GM 부등식과의 연결

```
원소가 고르게 분포 → 합이 상대적으로 큼 → 합² > 곱 → 1 반환
원소 중 큰 수 존재 → 곱이 급격히 커짐 → 곱 > 합² → 0 반환
```

- [3, 4, 5, 2, 1]: 곱=120, 합²=225 → 고른 분포 → 1
- [5, 7, 8, 3]: 곱=840, 합²=529 → 큰 수들의 곱이 지배 → 0
- 산술-기하 평균 부등식(AM-GM): 원소가 동일할 때 `합² / n² == 곱^(1/n)` (등호 성립)

## 📝 오늘의 패턴 요약

```
math.prod(iterable)            리스트 원소의 곱 (Python 3.8+)
int(a < b)                     비교 결과를 0/1 정수로 변환
sum(lst) ** 2                  합의 제곱 — sum 먼저, ** 나중
```

**핵심 솔루션:**
```python
def solution_v1(num_list: list[int]) -> int:
    """
    [Approach] math.prod로 곱, sum으로 합을 구한 뒤 비교 결과를 int로 변환
    [Time] O(n)  [Space] O(1)
    """
    from math import prod
    return int(prod(num_list) < sum(num_list) ** 2)
```

단 한 줄로 `곱 < 합²` 비교를 처리 — `math.prod`, `sum`, `int(bool)` 세 가지 Python 관용 패턴이 결합된 예시.
