## 오늘 배운 핵심 개념

### 1. Dictionary 활용법 (120883, 120886)

**오늘 깨달은 것:** Dictionary를 알고만 있었지 제대로 활용하지 못했음

```python
# 1. 안전한 조회 + 기본값
dic[c] = dic.get(c, 0) + 1  # KeyError 방지

# 2. 2차원 배열 → 딕셔너리 변환
db = [["id1", "pw1"], ["id2", "pw2"]]
db_dict = dict(db)  # {"id1": "pw1", "id2": "pw2"}

# 3. 딕셔너리 비교
dic1 == dic2  # 순서 무관, 키-값 쌍 모두 비교
Counter(s1) == Counter(s2)  # 애너그램 판별
```

→ `notes/dictionary-fundamentals.md`에 상세 정리

### 2. Counter 내부 동작 원리

```python
# ❌ O(N²) - count() 반복 호출
for c in string:
    dic[c] = string.count(c)  # 매번 전체 순회!

# ✅ O(N) - 누적 카운팅 (Counter 방식)
for c in string:
    dic[c] = dic.get(c, 0) + 1  # 해시 테이블 O(1) 연산
```

**핵심:** Counter가 빠른 이유는 해시 테이블의 O(1) 조회/갱신 덕분

### 3. 이진수 연산 (120885)

```python
# 내장 함수 활용
int("1010", 2)  # 10 (이진수 → 정수)
bin(10)         # "0b1010" (정수 → 이진수)
bin(10)[2:]     # "1010" (0b 제거)

# 수동 carry 연산 (면접 대비)
# LSB부터 처리: 문자열 뒤집기 → carry 처리 → 다시 뒤집기
```

### 4. 왈러스 연산자 `:=` (120887)

```python
# 기존 방식
l_str = str(l)
if k_str in l_str:
    cnt += l_str.count(k_str)

# 왈러스 연산자 (할당과 조건 동시에)
if k_str in (l_str := str(l)):
    cnt += l_str.count(k_str)
```

## 복습 필요 사항

### 120883 - dict(2차원배열) 패턴
- 2차원 배열을 딕셔너리로 변환하는 것 처음 알았음
- `dict([["a", 1], ["b", 2]])` → `{"a": 1, "b": 2}`

### 120885 - 이진수 수동 덧셈
- carry 연산 로직 복습 필요
- LSB(Least Significant Bit)부터 처리하는 이유 이해하기

### 120886 - 애너그램 판별 3가지 방법
- v1: 수동 딕셔너리 + count() → O(N²)
- v2: sorted() 비교 → O(N log N)
- v3: Counter 비교 → O(N) ← **최적**
