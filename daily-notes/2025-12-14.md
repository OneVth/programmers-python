## 오늘 배운 핵심 개념

### 1. dict.fromkeys() - 순서 보존 중복 제거 (120888)

```python
# ❌ set은 순서 보장 안 됨
list(set("abcabc"))  # ['c', 'a', 'b'] (순서 무작위)

# ✅ dict.fromkeys()는 삽입 순서 유지 (Python 3.7+)
"".join(dict.fromkeys("abcabc"))  # "abc"
```

**원리:** dict는 삽입 순서를 유지하고, 중복 키는 무시됨

→ `notes/dict-fromkeys.md`에 상세 정리

### 2. 문자열 대소문자 메서드 (120893)

```python
s = "hELLo"
s.swapcase()     # "HellO" - 대↔소 토글
s.capitalize()   # "Hello" - 첫 글자만 대문자
s.title()        # "Hello" - 단어별 첫 글자
s.casefold()     # "hello" - 강력한 소문자 (독일어 ß → ss)
```

**핵심:** `swapcase()`로 대소문자 토글 한 줄 해결!

→ `notes/str-case-methods.md`에 상세 정리

### 3. 튜플 키 정렬 - 다중 조건 (120890)

```python
# 가장 가까운 수, 같으면 작은 수 우선
sorted(array, key=lambda x: (abs(x - n), x))[0]
```

**원리:** 튜플은 첫 번째 요소로 비교 → 같으면 두 번째 요소로 비교

### 4. 문자열 슬라이싱 step 활용 (120892)

```python
cipher = "abcdefgh"
code = 2

# code번째, 2*code번째, 3*code번째... 문자 추출
cipher[code-1::code]  # "bdfh"
```

**핵심:** `[start:end:step]`에서 step만 잘 쓰면 반복문 없이 추출 가능

### 5. 다중 할당으로 스왑 (120895)

```python
# Python의 동시 할당
a, b = b, a  # 임시 변수 불필요!

# 리스트에서 인덱스 스왑
lst[i], lst[j] = lst[j], lst[i]
```

**원리:** 우변이 먼저 튜플로 패킹 → 좌변에 언패킹

### 6. Counter 순회 특성 (120896)

```python
from collections import Counter

counter = Counter("hello")  # {'l': 2, 'h': 1, 'e': 1, 'o': 1}

# 키만 순회
for c in counter:
    print(c, counter[c])

# 빈도 1인 문자만 추출
"".join(sorted(c for c in counter if counter[c] == 1))
```

### 120891 - 369게임 반례

```python
# ❌ 잘못된 풀이 - 0도 3으로 나누어 떨어짐!
sum(int(c) % 3 == 0 for c in str(order))  # "10" → 1 (오답)

# ✅ 올바른 풀이 - 정확히 3, 6, 9만 체크
sum(c in "369" for c in str(order))  # "10" → 0 (정답)
```
