### 2. re.split vs re.findall (문제 #120864)

```python
import re

s = "aAb1B2cC34oOp"

# 방법 1: 숫자가 아닌 것으로 split
re.split(r"\D+", s)     # ['', '1', '2', '34', '']  ← 빈 문자열 처리 필요

# 방법 2: 숫자만 findall (더 깔끔)
re.findall(r"\d+", s)   # ['1', '2', '34']
```

**교훈:** "제거 후 분리"보다 "원하는 것만 추출"이 더 간결할 때가 많음

### 3. 안전지대 — Set 좌표 추적 (문제 #120866)

```python
# 지뢰 주변 8방향을 Set에 추가 → O(1) 조회
danger = set()
for r, c in mines:
    for dr in (-1, 0, 1):
        for dc in (-1, 0, 1):
            danger.add((r + dr, c + dc))

# 안전 칸 = 전체 - 위험 칸
safe = sum(1 for r in range(n) for c in range(n) if (r, c) not in danger)
```
