### 2. Python 정수 나눗셈 `//` vs `/`

```python
7 / 2   # 3.5 (float)
7 // 2  # 3   (int, 내림)
-7 // 2 # -4  (음수는 더 작은 쪽으로 내림!)
```

## 💡 오늘의 깨달음

1. **환경 구축이 먼저**: runner.py와 표준 레이아웃을 먼저 만들어두니 이후 문제 풀이가 훨씬 빨라짐
