## 오늘 배운 핵심 개념

### 1. S+S 트릭 - 문자열 회전 판별 (120921)

```python
A = "hello"
B = "ohell"

# 핵심: B*2에 모든 회전 상태가 포함됨
# "ohellohell" 안에 "hello"가 있으면 회전 관계!

(B * 2).find(A)  # 1 → 오른쪽 1번 회전
```

**원리:**
```
B*2 = "ohellohell"
       ├─────┤      → "ohell" (0번)
        ├─────┤     → "hello" (1번) ← 찾음!
         ├─────┤    → "elloh" (2번)
```

→ `notes/string-rotation-trick.md`에 상세 정리

### 2. Replace vs S+S 패턴 구분법 (120956)

| 구분 | Replace 패턴 | S+S 트릭 |
|------|-------------|---------|
| **질문** | "조각들로 조립 가능?" | "회전해서 같아짐?" |
| **문자** | 사라짐 (제거) | 보존됨 (순서만 변경) |
| **길이** | 변할 수 있음 | 항상 동일 |

```python
# Replace 패턴 (옹알이)
word.replace("aya", " ")  # 조각 제거

# S+S 트릭 (문자열 밀기)
B in (A + A)  # 회전 확인
```

→ `notes/string-pattern-comparison.md`에 상세 정리

### 3. 정규표현식 패턴 분석 (120956)

```python
pattern = r'^(aya|ye|woo|ma)+$'

# 분해:
# ^        → 시작
# (...)    → 그룹
# a|b|c    → OR 연산
# +        → 1회 이상 반복
# $        → 끝
```

**match vs fullmatch:**
```python
re.match(r'^(aya)+$', word)   # ^$ 필요
re.fullmatch(r'(aya)+', word) # ^$ 불필요 (전체 매칭)
```

→ `notes/regex-tutorial.md`에 상세 정리

### 4. 2D 배열 대각선 인덱스 (181829)

```
     j=0  j=1  j=2
i=0   0    1    2    ← i+j 값
i=1   1    2    3
i=2   2    3    4
```

| 공식 | 의미 | 방향 |
|------|------|------|
| `i + j = k` | k번째 대각선 | ↙ |
| `i - j = k` | k번째 대각선 | ↘ |
| `i + j <= k` | 왼쪽 위 삼각형 영역 | |

→ `notes/2d-array-diagonal.md`에 상세 정리

### 5. DP 기저 조건 중요성 (120922)

```python
# ❌ min([])은 ValueError 발생!
dp[1][1] = min(candidates)  # candidates가 빈 리스트일 때 에러

# ✅ 기저 조건으로 빈 리스트 방지
if m == 1 and n == 1:
    dp[m][n] = 0
else:
    # candidates 계산...
    dp[m][n] = min(candidates)
```

### 120956 - 옹알이 반례 1: 경계 무시

```python
# ❌ permutation + in 사용
"maa" in "maayayewoo"  # True! (ma + a가 붙어서 maa처럼 보임)

# 실제로 "maa"는 유효하지 않은 옹알이
```

**교훈:** `in` 연산자는 부분 문자열을 찾지, 경계를 구분하지 않음

### 120956 - 옹알이 반례 2: 새로운 패턴 생성

```python
# ❌ 빈 문자열로 대체
"wyeoo".replace("ye", "")  # "woo" → 가짜 패턴 생성!

# ✅ 공백으로 대체 (경계 유지)
"wyeoo".replace("ye", " ")  # "w oo" → 경계 유지됨
```

**교훈:** Replace로 조각 검사할 때 반드시 **공백**으로 대체
