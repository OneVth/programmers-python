## 🎯 핵심 학습 내용

### 1. `set` 리터럴로 케이스 분기 — 주사위 게임 2

```python
# set 크기로 중복 개수를 판별
l = len({a, b, c})   # 1: 모두 같음 / 2: 두 개 같음 / 3: 모두 다름
```

- `set([a, b, c])` 대신 `{a, b, c}` — 리스트 생성 없이 바로 set 리터럴
- 세 조건의 공식은 **계층적 포함 관계**: 모두 다름 → ×(a²+b²+c²) → ×(a³+b³+c³)

```python
def solution_v1(a: int, b: int, c: int) -> int:
    l = len({a, b, c})
    if l == 1:
        return (a + b + c) * (a**2 + b**2 + c**2) * (a**3 + b**3 + c**3)
    elif l == 2:
        return (a + b + c) * (a**2 + b**2 + c**2)
    else:
        return a + b + c
```

### 2. `enumerate` + generator expression — 등차수열의 특정한 항만 더하기

```python
# i번째 항 = a + i * d, included[i]가 True인 항만 합산
sum(a + d * i for i, b in enumerate(included) if b)
```

- `enumerate(included)` → `(인덱스, bool)` 쌍을 동시에 순회
- generator expression은 리스트를 생성하지 않아 O(1) 공간 — `sum(list comp)` 대비 효율적
- `zip(range(n), included)` 대비 더 간결

```python
def solution_v1(a: int, d: int, included: list[bool]) -> int:
    return sum(a + d * i for i, b in enumerate(included) if b)
```

### 3. 상태 머신 패턴 — 코드 처리하기

```python
# "1" 문자가 mode를 0↔1로 전환, mode에 따라 짝수/홀수 idx 문자를 수집
mode = 0
for i, c in enumerate(code):
    if c == "1":
        mode ^= 1          # 0→1, 1→0 (XOR 비트 반전)
    elif i % 2 == mode:    # mode=0: 짝수 idx / mode=1: 홀수 idx
        ret.append(c)
```

- `i % 2 == mode` — mode와 idx 홀짝성이 일치할 때만 수집 (단축 조건)
- 문자열 누적은 `list.append()` + `"".join()` — `+=` 연결은 O(n²), join은 O(n)
- 빈 결과 처리: `return "".join(ret) if ret else "EMPTY"`

```python
def solution_v1(code: str) -> str:
    ret = []
    mode = 0
    for i, c in enumerate(code):
        if mode == 1:
            if c == "1":
                mode = 0
            elif i % 2 == 1:
                ret.append(c)
        else:
            if c == "1":
                mode = 1
            elif i % 2 == 0:
                ret.append(c)
    return "".join(ret) if ret else "EMPTY"
```
