### 2. 약수 구하기 - 제곱근 최적화 (120897)

```python
# v1: 조건문으로 제곱수 중복 방지
answer = []
for i in range(1, int(n**0.5) + 1):
    if n % i == 0:
        if i * i == n:
            answer.append(i)
        else:
            answer.append(i)
            answer.append(n // i)
return sorted(answer)

# v2: set으로 중복 자동 제거 ✨
answer = set()
for i in range(1, int(n**0.5) + 1):
    if n % i == 0:
        answer.add(i)
        answer.add(n // i)
return sorted(answer)
```

**원리:** 약수는 쌍으로 존재 (3×4=12) → √n까지만 확인하면 O(√n)

**v1 vs v2:**
- v1은 로직이 명시적 (왜 조건문이 있는지 보임)
- v2는 자료구조의 특성 활용 (버그 가능성 낮음)

### 3. 문자열 수식 계산 - 음수 변환 트릭 (120902)

```python
# v1: 뺄셈을 음수 덧셈으로 변환 ✨
exp = my_string.replace(" - ", " + -")
return sum(int(c) for c in exp.split(" + "))

# 예: "3 - 4 + 5" → "3 + -4 + 5" → sum([3, -4, 5]) = 4
```

**적용 조건:** +, - 연산자만 있고 우선순위가 동일할 때

**확장:** *, / 추가 시 → 후위 표기법(RPN) 또는 재귀 하강 파서 필요

### 4. find() vs enumerate - 인덱스 탐색 (120904)

```python
# v1: find() 내장 메서드
answer = str(num).find(str(k))
return answer + 1 if answer != -1 else -1

# v2: enumerate로 직접 순회 (early return)
for i, c in enumerate(str(num)):
    if c == str(k):
        return i + 1
return -1
```

**find() vs index():**
| 메서드 | 실패 시 | 예외 처리 |
|--------|---------|----------|
| `find()` | `-1` 반환 | 불필요 ✨ |
| `index()` | `ValueError` | try-except 필요 |

**주의:** 문제가 1-based 인덱스를 요구하면 `+1` 보정 필요
