### 1. 점진적 팩토리얼 계산 (문제 #120848)

**문제:** i! ≤ n인 최대 정수 i 찾기

```python
def solution(n: int) -> int:
    i, v = 1, 1
    while v < n:
        i += 1
        v *= i

    if v == n:
        return i
    return i - 1
```

**핵심 인사이트:**
- 팩토리얼은 급격히 증가 (10! = 3,628,800)
- 미리 계산하지 않고 **반복문 내에서 점진적으로 계산**
- 최대 10회 반복으로 O(1)에 가까운 성능

**경계 조건 처리:**
```python
# while 종료 시 v >= n
# v == n이면 정확히 i!이 n → i 반환
# v > n이면 i!이 n 초과 → i-1 반환
```

### 2. 좌표 정렬로 꼭짓점 위치 고정 (문제 #120860)

**문제:** 직사각형 4개 꼭짓점 좌표로 넓이 계산

```python
def solution(dots: list[list[int]]) -> int:
    sorted_dots = sorted(dots)
    # 정렬 후: [좌하, 좌상, 우하, 우상]
    return abs(sorted_dots[0][1] - sorted_dots[1][1]) * abs(
        sorted_dots[0][0] - sorted_dots[2][0]
    )
```

**정렬 후 인덱스 패턴:**
```
sorted(dots) 결과: [(x1,y1), (x1,y2), (x2,y1), (x2,y2)]
                    좌하     좌상     우하     우상

- [0]과 [1]: 같은 x → y 차이 = 높이
- [0]과 [2]: 다른 x → x 차이 = 너비
```

**대안적 접근:**
```python
# min/max 방식 - 더 직관적
xs, ys = zip(*dots)
return (max(xs) - min(xs)) * (max(ys) - min(ys))
```
