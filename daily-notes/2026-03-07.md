## 🎯 핵심 학습 내용

### 1. 1로 만들기 — 시뮬레이션 vs 수학 (문제 #181880)

```python
# 방법 1: while 시뮬레이션
answer = 0
for i in num_list:
    while i != 1:
        i //= 2
        answer += 1

# 방법 2: bit_length (수학적)
# 2^k → 1이 되려면 k번 나누기 = bit_length() - 1
sum(x.bit_length() - 1 for x in num_list)
```

**`bit_length()`의 의미:** 이진수 표현에 필요한 비트 수
- `1` → 1비트, `2` → 2비트, `4` → 3비트, `8` → 4비트
- 2로 나누는 횟수 = `bit_length() - 1`

### 2. 고정점(Fixed Point) 탐색 (문제 #181881)

```python
def solution(arr: list[int]) -> int:
    x = 0
    while True:
        changed = False
        for idx in range(len(arr)):
            val = arr[idx]
            if val >= 50 and val % 2 == 0:
                arr[idx] = val // 2
                changed = True
            elif val < 50 and val % 2 == 1:
                arr[idx] = val * 2 + 1
                changed = True
        if not changed:
            return x
        x += 1
```

**핵심 패턴:** `changed` 플래그로 수렴 감지
- 새 배열을 매번 생성하고 비교하는 것보다 in-place + 플래그가 메모리 효율적
- `while True` + `changed` 플래그 = 고정점 탐색의 관용 패턴

### 3. 조건부 배열 변환 (문제 #181882)

```python
# 181882는 한 번만 변환, 181881은 수렴할 때까지 반복
# 동일한 조건 분기지만 반복 여부가 다름
answer = []
for i in arr:
    if i >= 50 and i % 2 == 0:
        answer.append(i // 2)
    elif i < 50 and i % 2 == 1:
        answer.append(i * 2)
    else:
        answer.append(i)
```
