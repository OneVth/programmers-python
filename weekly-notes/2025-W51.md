## 💡 이번 주 핵심 패턴 정리

### 1. S+S 트릭 — 문자열 회전 판별 (120921)

```python
# B를 회전해서 A를 만들 수 있는지 확인
# 핵심: B+B에 모든 회전 상태가 포함됨
(B + B).find(A)  # 밀기 횟수 반환, 없으면 -1
```

**원리:**
```
"ohell" + "ohell" = "ohellohell"
                     ├─────┤ → "ohell" (0번 회전)
                      ├─────┤ → "hello" (1번 회전) ← 찾음!
```

### 2. 튜플 키 다중 정렬 (120880)

```python
# 거리 오름차순 → 값 내림차순
sorted(numlist, key=lambda x: (abs(x - n), -x))
```

**원리:** 튜플은 사전식(lexicographic) 비교

### 3. 교차 곱으로 기울기 비교 (120875)

```python
# 나눗셈 → 교차 곱 변환 (부동소수점 오차 방지)
(y2 - y1) * (x4 - x3) == (y4 - y3) * (x2 - x1)
```

### 4. 대각선 인덱스 패턴 (181829)

```python
# i + j = k: k번째 우상향 대각선
# i + j <= k: 좌상단 삼각 영역
for i in range(n):
    for j in range(m):
        if i + j <= k:
            answer += board[i][j]
```

### 5. 옹알이 — 공백 치환 패턴 (120956)

```python
# 빈 문자열 대체 → 잘못된 패턴 생성 위험!
"woowo".replace("woo", "")  # → "wo" (가짜 패턴)

# 공백 대체 → 경계 유지
"woowo".replace("woo", " ")  # → " wo" → strip 후 "wo" → 불가 판정
```

## ❌ 재복습 필요 분석

### 1. 120886 — Counter 아나그램 판별

```python
# 방법 1: 정렬 비교 O(n log n)
sorted(before) == sorted(after)

# 방법 2: Counter 비교 O(n) ← 최적
Counter(before) == Counter(after)
```

**체화 필요:** Counter가 빠른 이유 (해시 테이블 O(1) 연산)

### 2. 120888 — 순서 유지 중복 제거

```python
# dict.fromkeys() O(n) ← Python 3.7+ 순서 보장
"".join(dict.fromkeys(my_string))
```

**체화 필요:** `dict.fromkeys()` 패턴 자연스럽게 사용하기

### 3. 120891 — 369게임 0 처리

```python
# ❌ 0 % 3 == 0도 True!
int(c) % 3 == 0

# ✅ 정확히 3, 6, 9만
c in "369"
```
